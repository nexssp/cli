# List

Creates a folder with logos and json file with all the languages.

## Check what is in config

```sh
nexss py config # display config for python (.py extension)
```

## Format

```json
{
  "alt": details.title,
  "url": details.url,
  "extension": key,
  "logo": `${logoFileName}`,
  "years": details.years
}
```
